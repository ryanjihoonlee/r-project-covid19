# Exploratory Data Analysis on COVID-19 Dataset

## Overview
This project performs an exploratory data analysis (EDA) on the COVID-19 pandemic, tracing its evolution from an epidemic in Wuhan, China, to a global crisis. By examining the progression of confirmed cases worldwide, the analysis compares the spread within China to the rest of the world, highlighting key events and trends. The insights from this analysis provide a deeper understanding of the effectiveness of national policies and the overall trajectory of the pandemic.

## Objective
The primary aim is to utilize linear regression techniques and visualizations to uncover patterns in the spread of COVID-19, identifying significant trends and events that impacted the virus's transmission globally.

## Tools
The analysis was conducted using the R programming language, utilizing various statistical and data visualization tools.

Packages used:

#### dplyr
#### ggplot2
#### readr

## Dataset
The dataset includes confirmed COVID-19 cases worldwide, featuring specific data for cumulative confirmed cases globally, in China, and in other countries.

## Source
The data is available at https://github.com/RamiKrispin/coronavirus.
